# DevStash

A developer knowledge portal to store notes, snippets, commands, prompts, files, image, links and custom types.

## Context Files

The following files contain the context of the project: 

- @context/project-overview.md
- @context/coding-standards.md
- @context/ai-interactions.md
- @context/current-feature.md

## Commands

- `npm run dev` — Start dev server (Next.js with Turbopack)
- `npm run build` — Production build
- `npm run start` — Start production server
- `npm run lint` — Run ESLint (v9 flat config with next/core-web-vitals and TypeScript)
